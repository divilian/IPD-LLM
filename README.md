# IPD-LLM

Iterated Prisoner's Dilemma with a variety of Agent algorithms including
GGUF-compatible LLMs (for `llama.cpp`).

## Installation

```bash
# Install dependencies
pip install -r requirements.txt
```

## Usage

```bash
python pris.py -h
python pris.py 40 --agent-fracs Sucker 0.3 Mean 0.3 LLMvanilla 0.4
```
